RIPEN.CSS

RGOS FRAMEWORK

An initiative of the RGOS

ripen.css is a CSS framework for building interface components that look like the RGOS

This framework relies on the usage of semantic HTML. Certain elements can be directy used and others would be provided with a CSS class. For example, to make a button, you'll need to use a < button > ...., This page will guide you through that process, but for more info, you can check here.

You can override many of the styles of your elements while maintaining the appearance provided by this framework. This framework does not contain any JavaScript, it merely styles your HTML with some CSS. This means ripen.css is compatible with your frontend framework of choice.

Installation

The current way for installation is you can download the framework file from github releases. Soon we would implement other ways like npm and UNPKG.

Steps:
  1. For downloading the framework, please head to: The Releases
  2. After downloading, Link it to your HTML 
    <link rel="stylesheet" href="ripen.css">
  3. You are set to go!
And make sure to download the latest-stable version and enjoy!
